# Contribuciones

Gracias por tu interés en contribuir.

## Reportar Bugs

Abre un [issue](../../issues) con:
- Título claro del problema
- Pasos para reproducir
- Comportamiento esperado vs actual

## Pull Requests

1. Fork el repositorio
2. Crea rama: `git checkout -b feature/tu-feature`
3. Commit: `git commit -m "feat: descripción"`
4. Push: `git push origin feature/tu-feature`
5. Abre Pull Request

## Estilo de Código

- Sigue PEP 8
- Usa nombres descriptivos
- Agrega docstrings en funciones

## Documentación Médica

Solo información de fuentes profesionales:
- American Diabetes Association (ADA)
- European Association for the Study of Diabetes (EASD)
- Publicaciones médicas revisadas por pares
