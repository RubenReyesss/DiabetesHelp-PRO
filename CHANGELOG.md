# Changelog

## [1.0.0] - 2025-12-02

### Agregado
- Sistema RAG con 7 documentos médicos profesionales
- Integración con OpenRouter + Mistral-7B
- 5 pestañas funcionales: Perfil, Menú, Plan Salud, Chat, Info
- DiabetesTools con 5 funciones médicas (BMI, calorías, carbohidratos, insulina, tips)
- Chat con contexto completo del usuario
- Plan de salud adaptado al día de la semana
- Generación de menús personalizados
- Sistema de cálculos médicos (Mifflin-St Jeor)
- Documentación profesional completa
- GitHub Actions CI/CD
- MIT License con disclaimer médico

### Documentos Médicos
1. Diabetes Tipo 1 - Insulinas y manejo
2. Diabetes Tipo 2 - Medicamentos y epidemiología
3. Nutrición y Carbohidratos - Conteo y distribución
4. Monitoreo de Glucosa - HbA1c, CGM, emergencias
5. Complicaciones - Micro y macrovasculares
6. Medicamentos - Insulinas y agentes orales
7. Prevención - Factores de riesgo y DPP

### Tecnologías
- Gradio 4.x
- Python 3.8+
- Mistral-7B (OpenRouter)
- RAG System personalizado
